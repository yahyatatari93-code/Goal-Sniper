عرّاف الدوري الإنجليزي 🏆

منصة ذكية وممتعة لتوقع نتائج مباريات الدوري الإنجليزي الممتاز.
يتميز التطبيق بنظام حساب نقاط معقد يعتمد على مستوى الفرق المتواجهة، بالإضافة إلى ميزات إضافية تزيد من حماس المنافسة بين الأصدقاء.

🚀 المميزات الأساسية

لوحة تحكم خاصة: لإضافة المباريات وتسجيل النتائج الحقيقية.

نظام الدوريات المصغرة: إنشاء دوريات خاصة ومشاركة الكود مع الأصدقاء.

الأوسمة والإنجازات: (القناص، قاهر الكبار، عراف الأسبوع).

بطاقات القوة (Power-ups): كابتن، الجوكر الخطير (Triple Captain)، ودرع التأمين (Shield).

نظام قفل التوقعات: يُقفل التوقع تلقائياً قبل انطلاق المباراة بساعة واحدة.

حفظ البيانات: يعتمد التطبيق على التخزين المحلي (Local Storage) لسرعة الأداء.

🛠️ التقنيات المستخدمة

HTML5

Tailwind CSS (للتصميم المجاوب والعصري)

Vanilla JavaScript (للمنطق البرمجي)
